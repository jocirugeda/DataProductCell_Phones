Introduction

This Data Product explore the relationship between:
  • Cell Phone Subscriptions - By Country
  • GDP Per Capita at PPP By Country
  • Diferent country Regions ( Asia, Europe, etc )
The data comes from the web portal Quandl ( www.quandl,com ), we had to dowload data form two diferent Datasets:
  • GDP Per Capita at PPP By Country
  • Cell Phone Subscriptions - By Country
For download and process all the data , we create a Python script that uses MongoDB as data backend for processing.

The App dataset structure

  • ISO , standar code for the country
  • year, year of the related register
  • country.name, descriptive name of the country ( extracted from ISO codes)
  • region, diferent zones of countries ( the ones from quandl datasets )
  • Subscriptors.per100, number of Cell Subscribers per 100 Persons
  • GPDperCapita, Per Capita GDP based on PPP Valuation, USD. Units: Current international dollar.
  • years.with.network, computed value por each register ( year - first.year with subscribers data +1 ) this value shows how old is the Cell network in that country/year)

Aplication Help:

This App , have two diferent Tabs with information:

Explore Country Zones

Input Controls:
  • Region, is used to choose which Countries zone we want to explore
  • Plot type, this allow to choose which type of plot we want to see
    • Time Series, plot the whole history of data ( subscriptors by year)
    • Country Networks, this plots show the relationship between ( subscriptors by year with network) for the selected year
  • Network in year, used to filter years for the Country Networks plot, also filters the table of countries at the right-botton
Output Controls:
  • Plot, in this zone is displayed the selected plot
  • Summary, in this text-box is the output of the summary command of the data related to the plot that is above
    • type Time Series the stats are for year 2012
    • type Country Networks, the stats are for selected year
  • Table of Data, output of data filtered by year and Region

Regresion Models

Input Controls:
  • Region, is used to choose which Countries zone we want to explore
  • Network from year, used to filter years for the regresion model (years>x)
Output Controls:
  • Plot, in this zone is displayed the Residuals diagnostic plots
  • Summary, in this text-box is the output of the summary command of the computed model for the selected region filtered by year
